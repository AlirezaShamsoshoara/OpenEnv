action at obs, reward, done
G = ∑ γt · rt   (maximize cumulative reward)

● Agent

The learner. Takes observations and outputs actions. In LLM RL, this is the language model itself.

● Environment

The world. Receives an action, updates state, returns observation, reward, and done flag.

● Action

What the agent does. In agentic RL: tool calls, code execution, text responses. Sent via env.step(action).

● Observation

What the agent sees after acting. Contains reward (scalar feedback) and done (episode over?).

● Episode

One full interaction: reset() → multiple step() calls → done=True. Then start a new episode.
