1

Reset Environment

Start fresh episode. Environment returns initial observation.

obs = await env.reset()
2

Agent Decides

Model generates action from observation (LLM generates text/tool calls).

action = model.predict(obs)
3

Step Environment

Environment processes action, returns observation + reward + done.

result = await env.step(action)
4

Collect Trajectory

Store (state, action, reward) tuples for policy gradient computation.

5

Compute Loss

Use GRPO, PPO, or REINFORCE to compute policy gradient loss.

loss = grpo_loss(trajectory)
6

Update Model

Backpropagate and update weights. Repeat for next episode.
